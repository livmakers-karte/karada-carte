# からだのカルテ — デザイントークン（実装前に確定・自己批評済み）

案件固有の視覚アイデンティティ。「健康診断のカルテ／診断書」のメタファーを、日常の
からだの悩みに転用した総合メディア。核は **医療カルテの視覚言語（信頼性・精密さ・記録性・清潔感）**。
ありがちな「水色一辺倒のヘルスケア」も、editorial の「クリーム＋テラコッタ」も踏まない。

## コンセプト（1行）
> 主治医が発行する一枚の診断書のように、あなたのからだの「いま」を数値で記録するメディア。

## シグネチャー要素（記憶される唯一の独自要素）
**「カルテ結果票（診断票）」の発行体験。** 診断の最後に、カルテ番号・発行日・環境スコアの
ゲージ・原因タイプの**朱印（印泥色）**・主治医所見を持つ一枚の“診断書”が発行される。
朱印はサイト全体で唯一の暖色アクセントとして、この確定の瞬間に最も強く効かせる。
周囲（母屋・診断フロー）は静かな緑と用紙色で抑える＝大胆さを1箇所に集中。

## カラー（名前付き・6色＋2機能色）
医療現場の「深緑（信頼・生命・生薬/自然由来）× 診断用紙の温白 × 印泥の朱（記録・確定の印）」。
薄毛サイトに氾濫する〈クリニック水色〉〈メンズ黒×金〉〈before/after〉のいずれとも異なる。
Esrosso の「髪に悩んだ人間が作った」職人性・ノコギリヤシ等の植物由来とも意味的に接続する。

|---|---|---|
| `--pine`     | `#1E3A34` | 常磐に近い深緑。主構造色（ヘッダ/フッタ/見出し/濃地）＝主治医の信頼 |
| `--pine-2`   | `#2E5249` | 中緑。グラデ・所見帯 |

## タイポグラフィ（和文・実在フォント／外部CDN無し・システム同梱で完結）
カルテの精密さは「明朝の縦の理知＋数字の等幅」で出す。外部フォント配信はセキュリティ
ベースライン（外部JS/CDNを避けローカル化）に反するため、**システム同梱の明朝/ゴシックで構成**
（Windows=遊明朝/游ゴシック、Mac=ヒラギノ）。装飾は字体差でなく罫線・余白・朱印で作る。

- 見出し（Display）: `"Yu Mincho","Hiragino Mincho ProN",serif` — 診断書の格式・記録性
- 本文（Body）: `system-ui,"Hiragino Kaku Gothic ProN","Yu Gothic UI","Meiryo",sans-serif` — 可読性
- 数値（Score/カルテ番号）: `Georgia, "Times New Roman", serif` ＋ `font-variant-numeric: tabular-nums`
  — 検査値票のような等幅数字。ラテン小ラベル（MEDICAL RECORD 等）は字間を広く取る

## レイアウト
- 母屋（`/`）: 健康診断が増える**メディアハブ**。「いま使える診断」1枚を主役カード＋
  「これから広げる領域」タイル群。config.json のコンテンツ一覧で無限に増設。max 920px。
- `/usuge/`: ヒーロー→GEO直接回答→**5ステップ診断**→**カルテ結果票発行**→タイプ別Esrosso
  アプローチ→オファー（全額返金保証）→らくトク便/LINE 導線→再診断案内→しくみ→FAQ。max 860px。
- 診断票は方眼罫・カルテ番号・朱印・所見欄を持つ“紙面”。印刷/保存に耐える print CSS。

## モーション（subject に奉仕・抑制）
`prefers-reduced-motion` 尊重。ステップ切替の微フェード、スコアゲージの一度きりの伸長、
朱印の押印（scale+opacity 一回）。パララックス・自動再生・過剰演出はしない。

## 品質フロア
タップ44px/フォント16px、キーボードfocus可視（outline）、色コントラストAA、
CSP(meta)+frame-busting、画像は全てインラインSVG（外部ホットリンク皆無）。

## 自己批評（AIデフォルト検査 — 実装前ゲート）
- [x] クリーム#F4F1EA＋serif＋テラコッタ？→ **不採用**。地色は温白#FBFAF6（より白/冷）、
      暖色は茶でなく印泥朱、書体は高コントラストserif一辺倒でなく明朝×等幅数字。
- [x] 黒背景＋アシッド/バーミリオン1色？→ **不採用**。地は用紙、朱は黒地でなく緑×紙上の“印”。
- [x] 大数字＋小ラベル＋グラデのヒーロー？→ **不採用**。ヒーロー主役はカルテ図版（線画SVG）。
      スコアは結果票の中で「検査値」として出す＝意味に根ざす。
- [x] 意味のない 01/02/03 連番？→ 使わない。ステップ番号は「STEP n/5」で機能的にのみ。
- [x] 水色一辺倒ヘルスケア／メンズ黒金／before-after？→ いずれも回避。深緑×用紙×朱印は
      薄毛カテゴリで非典型。生薬/自然由来（Esrosso）と意味接続し差別化。
- [x] okane-carte（紺×金）の視覚リサイクル？→ **色・書体・モチーフを別系統に**（緑×用紙×朱印／
      明朝×等幅／カルテ票）。共有するのは構造（config駆動・GAS・インラインSVG・GEO）のみ。
